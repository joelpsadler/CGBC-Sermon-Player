# CGBC Collection Art Folder

Use this folder for optional curated collection artwork.

Folder rule:

```text
assets/collections/<collection-key>/square.jpg
assets/collections/<collection-key>/wide.jpg
```

Example for RSS metadata:

```text
Collections: Spiritual Gifts, Church Body
```

Expected folders:

```text
assets/collections/spiritual-gifts/square.jpg
assets/collections/spiritual-gifts/wide.jpg
assets/collections/church-body/square.jpg
assets/collections/church-body/wide.jpg
```

Recommended art sizes:

- square.jpg: 1000x1000 or 1200x1200
- wide.jpg: 1280x720 or 1600x900
- Keep files small, ideally under 1 MB each.

If artwork is missing, the site falls back to episode/series art and then the generic fallback image.
